Detailed Description

A smart-pointer for managing an SPI_lru_cache even when it is only forward declared.

Automaticlly allocated cache with new, and assigns m_spi_lru_cache to it, when dereferenced using non-const operator->(). Destructor deletes the object pointed to by m_spi_lru_cache.
