)
static

Allocate gets a struct item from the slab allocator, and fills in everything but the value.

It seems like we can just pass this on to the default engine; we'll intercept it later in store(). This is also called directly from finalize_read() in the commit thread.

Parameters
handlein: Engine handle
cookiein: connection cookie
itemout: item to allocate
keyin: key value(s)
nkeyin: key length
nbytesin: value length
flagsin: flag
exptimein: expiration time
