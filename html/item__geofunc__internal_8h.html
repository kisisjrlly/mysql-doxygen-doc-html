◆ simplify_multi_geometry()

bool simplify_multi_geometry ( String *  str,
String *  result_buffer 
)

Simplify multi-geometry data.

If str contains a multi-geometry or geometry collection with one component, the component is made as content of str. If str contains a nested geometry collection, the effective concrete geometry object is returned.

Parameters
strA string buffer containing a GEOMETRY byte string.
[out]result_bufferif not NULL and if a simplification is to be done, we will copy the GEOMETRY byte string from str into result_buffer and simplify the one stored in result_buffer, and the one in str is intact. If there is any data in result_buffer before calling this function, it is overwritten; If no simplification done, the result_buffer is intact if it is provided.
Returns
whether the geometry is simplified or not.
