Detailed Description

Test component for the Performance Schema Notification service.

Upon installation, this component registers 5 callback functions for each notification event:

thread_create thread_destroy session_connect session_disconnect session_change_user

These events are triggered externally, e.g. from an MTR script. Each callback functions logs a message to a predefined logfile and to stderr.
