56 /**
57  * @brief DynamicState represents a MySQLRouter dynamic state object.
58  *
59  * It's ment to be used as a singleton that provides methods to read/update
60  * sections from the specific modules requiring saving their runtime state.

63  *
64  */

87  bool load();
