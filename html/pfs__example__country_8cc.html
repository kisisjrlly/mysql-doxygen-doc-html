bool is_duplicate ( Country_record *  record,
int  skip_index 
)

Check for duplicate value of Primary/Unique Key column(s).

A sequential search is being used here, but its upto plugin writer to implement his/her own search to make sure duplicate values are not inserted/updated for Primary/Unique Key Column(s).

Parameters
recordrecord to be checked for duplicate
skip_indexelement at this index not to be considered for comparison
Returns
true if duplicate found, false otherwise
