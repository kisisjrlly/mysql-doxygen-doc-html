Tests if handshake response packet has PROTOCOL_41 flag set.

This is a very simple method, it only checks that single flag and does nothing else (in particular, it doesn't perform any kind of validation)

void mysql_protocol::HandshakeResponsePacket::Parser41::parse ( Capabilities::Flags  server_capabilities)
overridevirtual

Parses handshake response packet.

This method assumes that the current packet is a PROTOCOL41 handshake response.

Parameters
server_capabilitiesCapability flags of the server. Client's flags will be &-ed with them before applying rules for packet parsing.
Exceptions
std::runtime_erroron unrecognised or invalid packet
