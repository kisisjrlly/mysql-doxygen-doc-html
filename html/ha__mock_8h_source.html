40 /**
41  * The MOCK storage engine is used for testing MySQL server functionality
42  * related to secondary storage engines.
43  *
44  * There are currently no secondary storage engines mature enough to be merged
45  * into mysql-trunk. Therefore, this bare-minimum storage engine, with no
46  * actual functionality and implementing only the absolutely necessary handler
47  * interfaces to allow setting it as a secondary engine of a table, was created
48  * to facilitate pushing MySQL server code changes to mysql-trunk with test
49  * coverage without depending on ongoing work of other storage engines.
50  *
51  * @note This mock storage engine does not support being set as a primary
52  * storage engine.
53  */
