26 /*
27  For collation which changes character's primary weight according to its
28  tailoring rule, we give this character a extra collation element (see
29  comments in my_char_weight_put_900()). Usually the primary weight of this
30  extra CE starts from 0x54A4, which is the biggest primary weight of all
31  regular characters (non-CJK and non-ignorable) in DUCET. But Chinese is
32  special, because to keep assigning single primary weight to character
33  groups like Latin, Cyrillic, etc, we used all weight value in [0x1C47,
34  0xF643], so we give the primary weight of extra CE starting from 0xF644
35  to avoid weight overlapping.
36  */
37 constexpr int ZH_EXTRA_CE_PRI = 0xF644;
