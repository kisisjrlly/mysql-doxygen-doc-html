Detailed Description

Creates a MySQL error packet.

This class creates a MySQL error packet which is send to the MySQL Client.

Constructor.

Note
The default constructor will set the error code to 1105, message "Unknown error", and SQL State "HY000". These values come from the MySQL Server server errors.
