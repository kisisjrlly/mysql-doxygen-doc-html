◆ version_token_check()

static int version_token_check ( MYSQL_THD  thd,
mysql_event_class_t  event_class,
const void *  event 
)
static

Audit API entry point for the version token pluign.

Plugin audit function to compare session version tokens with the global ones. At the start of each query (MYSQL_AUDIT_GENERAL_LOG currently) if there's a session version token vector it will acquire the GET_LOCK shared locks for the session version tokens and then will try to find them in the global version lock and compare their values with the ones found. Throws errors if not found or the version values do not match. See parse_vtokens(). At query end (MYSQL_AUDIT_GENERAL_STATUS currently) it releases the GET_LOCK shared locks it has aquired.

Parameters
thdThe current thread
event_classaudit API event class
eventpointer to the audit API event data
