221  * observers on replicaset status changes.

272  * Parameters host, port, user, password are used to setup the connection with
273  * the metadata server.

276  *

278  * connected to the metadata server.
279  *

281  * initialized.
282  *
283  * @param cluster_type type of the cluster the metadata cache object will
284  * represent (GR or ReplicaSet)
285  * @param router_id id of the router in the cluster metadata
286  * @param cluster_type_specific_id (id of the replication group for GR,
287  * cluster_id for ReplicaSet)
288  * @param metadata_servers The list of cluster metadata servers
289  * @param user_credentials MySQL Metadata username and password
290  * @param ttl The time to live for the cached data
291  * @param auth_cache_ttl TTL of the rest user authentication data
