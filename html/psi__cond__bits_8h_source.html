106 /**
107  State data storage for @c start_cond_wait_v1_t.
108  This structure provide temporary storage to a condition locker.
109  The content of this structure is considered opaque,
110  the fields are only hints of what an implementation
111  of the psi interface can use.
112  This memory is provided by the instrumented code for performance reasons.
113  @sa start_cond_wait_v1_t
114 */
