Running SASL authentication step till authentication process is concluded MySQL server plug-in working as proxy for SASL / LDAP server.

Kerberos authentication is concluded by the LDAP/SASL server, From client side, authentication is succeded and we need to send data to server side to conclude the authentication. Other SASL authentication are conculded in the client side.
