bool  for_new_connections = true 
) const
private

Gets available destinations from Metadata Cache.

This method gets the destinations using Metadata Cache information. It uses the metadata_cache::lookup_replicaset() function to get a list of current managed servers. Bool in the returned pair indicates if (in case of the round-robin-with-fallback routing strategy) the returned nodes are the primaries after the fallback (true) or secondaries (false).

The Metadata Cache to use.

cache_name_ is the the section key in the configuration of Metadata Cache.

For example, given following Metadata Cache configuration, cache_name_ will be set to "ham": 

[metadata_cache.ham]
