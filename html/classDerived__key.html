Detailed Description

This structure is used to keep info about possible key for the result table of a derived table/view.

The 'referenced_by' is the table map of tables to which this possible key corresponds. The 'used_field' is a map of fields of which this key consists of. See also the comment for the TABLE_LIST::update_derived_keys function.
