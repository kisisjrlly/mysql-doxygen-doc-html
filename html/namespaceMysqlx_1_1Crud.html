Detailed Description

Basic CRUD operations.

The CRUD operations work in a similar fashion as the SQL statements below:

  • Mysqlx::Crud::Insert - INSERT INTO ...
  • Mysqlx::Crud::Find - SELECT ... FROM ...
  • Mysqlx::Crud::Update - UPDATE ... SET ...
  • Mysqlx::Crud::Delete - DELETE FROM ...

Also operation related to manipulation of views:

  • Mysqlx::Crud::CreateView - CREATE VIEW ... AS ...
  • Mysqlx::Crud::ModifyView - ALTER VIEW ... AS ...
  • Mysqlx::Crud::DropView - DROP VIEW ...
