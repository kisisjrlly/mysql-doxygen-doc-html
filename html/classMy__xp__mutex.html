Detailed Description

Abstract class used to wrap mutex for various implementations.

A typical use case is:

My_xp_mutex *mutex= new My_xp_mutex_impl();
mutex->init(mutex_PSI_key, NULL);
mutex->lock();
...
mutex->unlock();
