Detailed Description

A stateless class that provides the implementation of the UDFs which test the Character set extension argument.

It determines the charset or collation names of the second UDF argument and use that to convert either return value or first UDF argument.

The udf implementation that tests the change in the character set of return value.

It picks the first UDF argument as return value then it converts the return value into the charset as specified during init() time. It uses charset converter service to do the actual conversion.
