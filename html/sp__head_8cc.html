Merge contents of two hashes representing sets of routines used by statements or by other routines.

Parameters
dsthash to which elements should be added
srchash from which elements merged
Note
This procedure won't create new Sroutine_hash_entry objects, instead it will simply add elements from source to destination hash. Thus time of life of elements in destination hash becomes dependant on time of life of elements from source hash. It also won't touch lists linking elements in source and destination hashes.
