static

Convert and/or validate input_string according to charset 'to_cs'.

If input_string needs conversion to 'to_cs' then do the conversion, and verify the result. Otherwise, if input_string has charset my_charset_bin, then verify that it contains a valid string according to 'to_cs'.

Will call my_error() in case conversion/validation fails.

Parameters
input_stringstring to be converted/validated.
to_csresult character set
output_string[out] output result variable
Returns
nullptr in case of error, otherwise pointer to result.
