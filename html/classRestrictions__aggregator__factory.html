Detailed Description

Factory class that solely creates an object of type Restrictions_aggregator.

  • The concrete implemenations of Restrictions_aggregator cannot be created directly since their constructors are private. This class is declared as friend in those concrete implementations.
  • It also records the CURRENT_USER in the binlog so that partial_revokes can be executed on slave with context of current user

A factory method that creates objects from Restrictions_aggregator hierarchy.

Creates an object iif –partial_revokes system variable is ON. It also records the CURRENT_USER in the binlog so that partial_revokes can be executed on slave with context of current user
