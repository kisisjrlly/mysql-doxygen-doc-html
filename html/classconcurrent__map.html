Detailed Description

template<typename Key, typename Value, typename Hash = std::hash<Key>>
class concurrent_map< Key, Value, Hash >

Basic Concurrent Map.

The concurrent_map is a hash-map, with fixed number of buckets. The numer of buckets can be specified in constructor parameter (num_buckets), by default is set to 23.
